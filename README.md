Sitemap & Search Engine Submission

1) Verify site ownership
- Google: Open Google Search Console (https://search.google.com/search-console) and add `https://prajwalshrestha2.com.np`.
- Bing: Open Bing Webmaster Tools (https://www.bing.com/webmasters) and add your site.

2) Submit sitemap
- In Search Console -> Indexing -> Sitemaps, enter: `https://prajwalshrestha2.com.np/sitemap.xml` and submit.
- In Bing Webmaster Tools -> Sitemaps, submit the same URL.

3) Quick pings (optional)
Run these from a machine with curl available:

```bash
curl "http://www.google.com/ping?sitemap=https://prajwalshrestha2.com.np/sitemap.xml"
curl "http://www.bing.com/ping?sitemap=https://prajwalshrestha2.com.np/sitemap.xml"
```
